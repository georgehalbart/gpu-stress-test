# GPU Benchmark & Stress Test

A lightweight browser-based GPU testing tool that allows you to run real-time stress tests and performance benchmarks using WebGL.

---

## 🚀 Features

- Real-time GPU stress testing  
- WebGL-based rendering engine  
- FPS monitoring and performance tracking  
- Multiple test modes (Particles, Triangles, Fractal, Plasma)  
- Adjustable intensity levels  
- Built-in benchmark scoring system  

---

## 📊 What It Does

This tool pushes your GPU using advanced WebGL rendering techniques to simulate high-load scenarios. It helps detect:

- GPU overheating  
- Performance drops (low FPS)  
- Stability issues  
- Hardware limitations  

---

## 🖥️ How to Use

1. Open the extension  
2. Select test mode  
3. Adjust intensity  
4. Click **Start Test** or **Run Benchmark**  
5. Monitor FPS and performance results  

---

## ⚠️ Warning

Running a GPU stress test may:
- Slow down your system  
- Cause lag or temporary freezing  

👉 Always save your work before testing
